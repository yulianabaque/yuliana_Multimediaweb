Sobre esta experiencia

Esta web no está pensada para usarse de una sola forma. Hay personas que entran con una emoción clara, y otras que no. Algunas solo escuchan música. Otras exploran diferentes estados.

Aquí no hay decisiones correctas. Solo emociones.

Elige cómo te sientes

Selecciona una emoción para comenzar la experiencia.

Un momento para parar

No hace falta interactuar constantemente. Puedes cambiar de emoción, bajar el volumen o simplemente dejar la música.

Esta web no guarda datos ni analiza emociones.

Solo cambia contigo.
