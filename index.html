Sobre esta experiencia

Esta web no está pensada para ser usada de una sola forma. Hay personas que entran con una emoción clara, y otras que no. Algunas solo escuchan música. Otras miran, leen o cambian varias veces.

Aquí no hay decisiones correctas. Solo estados.

Un momento para parar

No hace falta interactuar todo el tiempo. A veces solo quedarse aquí un rato ya es suficiente.

Puedes cambiar de emoción, bajar el volumen o simplemente dejar la música.

Esta web no guarda datos. No analiza emociones. No intenta entenderte.
